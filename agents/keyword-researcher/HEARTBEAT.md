# Heartbeat — Keyword Researcher

## 1. Check for Tasks

- New keyword research requests from CEO
- Product analyses from Produkt-Analyst (trigger for research)
- Refresh requests for existing keyword strategies

## 2. Read Input

- `produkt-analyse.md` — Product category, USPs, target audience, competitors

## 3. Research Process

### Step 3a: Seed Keywords
- Extract core terms from product name, category, and USPs
- Identify the primary search intent (what would the buyer type?)

### Step 3b: Expand
- Synonyms and related terms
- Long-tail variations (3-5 word phrases)
- Question-based queries (Rufus/COSMO optimization)
- Competitor reverse-engineering (what do top 5 rank for?)
- Seasonal/trend variations

### Step 3c: Qualify
For each keyword, determine:
- Search volume (estimated monthly)
- Competition level (High/Medium/Low)
- Purchase intent (Informational/Navigational/Transactional)
- Relevance to product (Primary/Secondary/Marginal)

### Step 3d: Tier & Prioritize

| Tier | Definition | Placement |
|---|---|---|
| **Primary (5-10)** | Highest relevance + volume + purchase intent | Title + Bullet Point 1 |
| **Secondary (10-20)** | Good relevance, moderate volume | Bullet Points 2-5 |
| **Long-tail (20-50)** | Low competition, high conversion potential | Bullets + Description |
| **Backend (remaining)** | Synonyms, misspellings, alternatives | Backend Search Terms |
| **PPC-only** | High volume but listing-irrelevant | PPC campaigns only |
| **Negative** | Irrelevant terms to exclude from PPC | Negative keyword list |

### Step 3e: Placement Plan

Map every primary and secondary keyword to its ideal placement:
- Title (position 1-3 for primary)
- Bullet Points (which bullet?)
- Description
- Backend Search Terms
- PPC Campaign Type (exact/phrase/broad)

## 4. Write Output

Write `keyword-strategie.md` to the task workspace.

## 5. Self-Check

- [ ] All keywords have: volume, competition, intent, relevance
- [ ] Clear tier assignment (Primary/Secondary/Long-tail/Backend/PPC-only)
- [ ] Placement plan for every Primary and Secondary keyword
- [ ] Question-based keywords included (Rufus optimization)
- [ ] Competitor keywords analyzed
- [ ] Negative keyword list provided for PPC
- [ ] No keyword stuffing risk (max 2-3 primary keywords in title)
