# Tools — ClickUp Manager

## Primary Tool

**ClickUp** — Your main workspace. You create, update, and track all tasks here.

Available ClickUp operations:
- Create/update/delete tasks
- Set task dependencies and links
- Create lists, folders, documents
- Track time entries
- Search tasks and workspace
- Manage reminders
- Send chat messages
- Filter tasks by status, assignee, due date

## Skills

- **ClickUp Automation** — Workspace management, task workflows, time tracking, team productivity
