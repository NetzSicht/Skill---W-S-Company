# Heartbeat — CEO

Run this checklist every heartbeat.

## 1. Check Inbox

- New client requests or briefs
- Completed tasks from agents (review and route)
- Escalations or blockers from agents
- Revision results from Quality-Reviewer

## 2. Triage New Work

For each new request, determine:

| Question | Action |
|---|---|
| Full listing optimization? | Create tasks for: Review-Analyst, Keyword Researcher, Produkt-Analyst (parallel). Then: Content Master, Listing-Briefer (after research). Then: Quality-Reviewer, A+ Content Designer. |
| Image briefing only? | Trigger Bild-Pipeline: Produkt-Analyst → Listing-Briefer → Quality-Reviewer |
| Text optimization only? | Trigger Text-Pipeline: Keyword Researcher → Content Master |
| PPC campaign? | Assign to PPC Specialist (with keyword data from Keyword Researcher if needed) |
| A+ Content? | Assign to A+ Content Designer (needs completed Bild-Pipeline + Text-Pipeline as input) |
| Review analysis? | Assign to Review-Analyst |

## 3. Monitor Active Pipelines

Check status of running tasks:
- Are any agents blocked? → Resolve or reassign
- Are any tasks overdue? → Follow up
- Did the Quality-Reviewer reject something? → Ensure revision is in progress

## 4. Route Completed Work

When an agent completes a task:
- Does it feed into the next pipeline step? → Route to next agent
- Is it a final deliverable? → Review and prepare for client
- Did Quality-Reviewer approve? → Package for delivery

## 5. Client Communication

- Prepare status updates for active projects
- Deliver completed work packages
- Flag any delays or scope changes proactively

## Full Listing Optimization — Pipeline Overview

```
Phase 1 (parallel):
  Review-Analyst ──────┐
  Keyword Researcher ──┤── feeds insights to Phase 2
  Produkt-Analyst ─────┘

Phase 2 (parallel, after Phase 1):
  Content Master (uses: keywords + review insights + product data)
  Listing-Briefer (uses: product analysis + review insights)

Phase 3 (after Phase 2):
  Quality-Reviewer (checks: briefing AND text)

Phase 4 (after Phase 3 approved):
  A+ Content Designer (uses: all approved outputs)

Phase 5:
  CEO packages everything for client delivery
```
